# This is a fully vibecoded mod using Grok 4.2 and Composer 1.
# See the mod at Modrinth!

Logic is a lightweight client-side utility mod for Minecraft 1.21.4 Fabric that helps you get better FPS with minimal effort.
Features:

Smart Boost — Analyzes your CPU, RAM, and GPU then automatically applies the best vanilla settings for your hardware.
Performance HUD — Clean, customizable FPS overlay with 1s/5s/10s averages, CPU usage, and GPU name.
One-Click Performance Pack — Installs the best performance mods (Sodium, Lithium, Starlight, FerriteCore, Entity Culling, ImmediatelyFast, C2ME, Krypton, ModernFix) directly from the dashboard.
Performance Dashboard (default F8) — Quick presets, hardware info, mod compatibility status, and profile export/import.
First-launch optimization — Automatically applies good settings on your first time launching with the mod.
Install Fabric Loader + Fabric API first.
Use a launcher like Prism Launcher or the Modrinth App for easy installation.
For shaders while keeping high FPS, add Iris (with Indium for Fabric Rendering API compatibility).
On Forge/NeoForge, look for ports like Embeddium (Sodium equivalent) and Radium (Lithium equivalent).
Optional extras: Noisium (faster worldgen), Enhanced Block Entities, LazyDFU (faster startup), or Nvidium (extra boost for NVIDIA GPUs).
Combine with vanilla optimizations: lower render distance, disable VSync, use a performance texture pack, and allocate 4–8 GB RAM.
